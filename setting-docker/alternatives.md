## Alternatives if You Can't Create an AWS Account

- **Docker Desktop** (Windows, macOS, Linux)
  - Recommended for most users
  - Easy to install and use

- **Oracle VirtualBox**
  - Run a complete virtual machine locally
  - Available on Windows, macOS, and Linux

- **VMware Workstation**
  - Another virtualization platform
  - Available on Windows and Linux

- **And do allthing on aws** 

> **Note:** You only need **one** of these options. Docker Desktop is the recommended choice for learning Docker.
