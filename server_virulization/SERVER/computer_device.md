# Every computer has many components, but the most basic ones are:

- **Central Processing Unit (CPU) – The Brain**
  - Involved in all operations that run on the computing device. => engine process everything

- **Random Access Memory (RAM)**
  - Used to run applications and hold data during processing. where apps are open , excuted while we are using them 

- **Hard Disk Drives (HDD) and Solid-State Drives (SSD)**
  - Provide persistent storage for the operating system, data, and applications. storage in short

- **Network Interface Cards (NIC)**
  - Connect the computing device to the outside world.
  - Can be wireless (Wi-Fi) or wired using Ethernet/LAN cables.
- transmission or reception device basically internet transfering

- **Graphics Processing Unit (GPU) / Graphics Card**
  - Accelerates video operations such as gaming and rendering.
  - Can be built into the motherboard or installed as a dedicated graphics card.
- additional cpus that can do specialized fn and speed up what you don computer.
